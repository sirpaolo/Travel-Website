# GABAi: Your Travel Guide Buddy 

GABAi is a web-based travel booking platform designed to showcase the beauty of the Philippines while providing users with a simple, secure, and user-friendly way to explore destinations and book accommodations and activities.

This project was developed as part of the **Web Design and Development Lab (T-CPET418LA)** and demonstrates full-stack web development skills, API integration, and database-driven booking systems.

---

##  Features

- **User Authentication**
  - Email and password registration
  - Google OAuth login

- **Destination Search**
  - Search for hotels, destinations, and activities using the TripAdvisor API

- **Curated Deals & Destinations**
  - Handpicked travel experiences with images, descriptions, and ratings

- **Booking System**
  - Select dates and number of guests
  - Review booking details before payment

- **Secure Payments**
  - Integrated Stripe API for secure card payments

- **Booking Confirmation**
  - Successful bookings are saved to the database
  - Confirmation page shown after payment

- **Responsive Design**
  - Optimized for desktop, tablet, and mobile devices
